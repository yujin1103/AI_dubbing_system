다국어 자동 더빙 시스템
영어 영상을 한국어로 자동 더빙하고 입모양까지 동기화하는 AI 파이프라인
Pipeline
1 작품명
다국어 자동 더빙 시스템 — 한국어 Voice Cloning + Lip-sync
2 작품 개요
영어 영상을 입력받아 한국어 더빙 영상을 자동 생성한다. ffmpeg과 BS-Roformer로 vocal과 BGM을 분리하고, Silero VAD로 silence를 제거한 뒤, Qwen3-ASR로 받아쓰기, DiariZen + ECAPA-TDNN으로 화자를 분리한다. LightASD와 emotion2vec+로 영상 속 화자·감정을 인식하고 GPT 5.4가 lip-sync syllable 가이드에 맞춰 한국어로 번역한다. CosyVoice3의 zero-shot voice cloning으로 화자별 한국어를 합성한 뒤, LatentSync 1.6 + Korean LoRA로 입모양을 맞추고 GFPGAN으로 화질을 복원한다. Docker 멀티 venv 4개 + daemon 3개 구조로 의존성 격리와 메모리 상주를 동시에 달성했다.
3 주요 적용 기술 및 구조
개발 도구Docker Desktop · WSL2 · VS Code · Git / GitHub · npm (mermaid-cli, pptxgenjs, puppeteer)
개발 언어Python 3.12 · Bash · JavaScript (Node.js) · CUDA 13.0
개발 환경RTX 5080 16GB sm_120 (Blackwell) · Docker container 56GB · 멀티 venv 4개 · daemon 3개 (port 8901/8902/8903)
주요 기술PyTorch 2.11+cu130 · PEFT 0.10 LoRA r=32 · 8-bit AdamW (bitsandbytes) · ECAPA-TDNN 임베딩 · ffmpeg loudnorm EBU R128 · DDIM 20-step · GFPGAN async I/O · ECAPA sliding window split
사용 모델CosyVoice3 (Fun-CosyVoice3-0.5B) · Qwen3-ASR-1.7B · DiariZen · LightASD · emotion2vec+ Large · LatentSync 1.6 + Korean LoRA · GFPGAN v1.4 · UTMOS-style MOS · GPT 5.4
표준·데이터AIHub 538 (립리딩 588영상 50k step) · EBU R128 (-23 LUFS) · 24kHz audio · 25fps video
출력한국어 더빙 mp4 + JSON 리포트 (segment별 화자/감정/번역/MOS)
